## 🚀 How to Use

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required
- Simple & lightweight projects

### Getting Started

1. **Clone this repository:**
   ```bash
   git clone https://github.com/aditya-rega-safi/web-programming-portfolio.git
   cd web-programming-portfolio
   ```

2. **Navigate to a project:**
   ```bash
   cd projects/1-personal-portfolio
   ```

3. **Open in browser:**
   - Double-click `index.html`, OR
   - Right-click → Open with → Your browser

4. **Test responsiveness:**
   - Open DevTools (F12)
   - Toggle Device Toolbar (Ctrl+Shift+M)
   - Test on mobile, tablet, desktop sizes

## 📊 Responsive Breakpoints Used

All projects follow mobile-first responsive design:

```css
/* Mobile First */
/* Styles for small screens (< 480px) */

/* Tablet */
@media (min-width: 768px) {
  /* Styles for medium screens (≥ 768px) */
}

/* Desktop */
@media (min-width: 1024px) {
  /* Styles for large screens (≥ 1024px) */
}
```
